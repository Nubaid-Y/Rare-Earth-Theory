# The Rare Earth Theory v1.0.0

The Rare Earth Theory v1.0.0 is a simple, terminal-based Python game that tries to simulate how challenging it is to create a planet that is capable of intelligent life, let alone complex life. This is the very first version of the project and is fully text-driven (due to my beginner level skill). There is no graphical interface. All interaction happens through the terminal using print statements and user input.

The goal of the game is to design a star, create a planet around it, and see whether your decisions result in a habitable world.

“...because you won the lottery, and most people will never even know there was a lottery.”
- Neil deGrasse Tyson


## Installation

Python 3.8 or newer is recommended.


## Features in v1.0.0

Fully terminal-based gameplay

Star creation system

Planet size selection

Planet lifespan selection

Basic habitability logic

Text-only output using print statements

## How to Run

Make sure Python is installed

Open a terminal

Follow the printed prompts in the terminal
